Tag radial-axis

  • Description:
    The radial tag is used to define a radial-axis. A chart can have more than one radial-axis positioned at different angles.
  • Component Information 
    Info Value
    Component Type org.javaq.chartfaces.chart.part.axis.radial
    Display Name None
    Handler Class None
    Renderer Type None
    Description None
  • Attributes 
    Name Required Type Description
    id false java.lang.Object ID of the axis. Must be unique in the page.
    angle false java.lang.Object The angle of the radial axis line. The unit is in degrees.
    caption false java.lang.Object Caption of the axis.
    captionStyle false java.lang.Object Style of the caption of the axis.
    gridLineDensity false java.lang.Object Density of the grid-lines of the axis. The value equals the number of ticks in between two grid lines, that is, if this value is 0, there will be exactly one grid line on each tick. Default value is 0.
    gridLineStyle false java.lang.Object The style of the grid lines.
    label false java.lang.Object Labels of the axis ticks.
    labelAngle false java.lang.Object Labels of the axis can be angled and this attribute defines the angle of the labels in degrees.
    labelStyle false java.lang.Object The style of the labels.
    lineStyle false java.lang.Object The style of the axis line.
    tickDirection false java.lang.Object The direction of the ticks. the tick direction. The tick direction can be clockwise or anti-clockwise, or middle.
    tickHeight false java.lang.Object The height of the tick in the chart viewBox coordinator.
    tickStyle false java.lang.Object The style of the ticks.
    title false java.lang.Object The mouse over hint text of the axis.
    max false java.lang.Object The max value of the axis. This is the real coordinate max value (as opposed to the viewBox coordinate).
    min false java.lang.Object The min value of the axis. This is the real coordinate min value (as opposed to the viewBox coordinate).
    scaler false java.lang.Object A scaler is used to scale a an axis to create, e.g., a log scale.
    value false java.lang.Object The axis tick values.
