  • Tag Summary 
    Tag Description
    bar The bar tag is used to define a bar chart. A bar chart can be laid out as either horizontal or vertical. A bar chart may contain more than one barset, each bound to one data collection.
    scatter The scatter tag is used to define a scatter chart. A scatter chart can be used to render 2-D, 3-D even 4-D data values by varying the size, and color of the "dots". A scatter chart may contain more than one scatterset, each bound to one data collection.
    pie The pie tag is used to define a pie chart.
    xaxis The xaxis tag is used to define a x-axis. A chart can have more than one x-axis positioned at different positions.
    yaxis The yaxis tag is used to define a y-axis. A chart can have more than one y-axis positioned at different positions.
    angular-axis The angular-axis tag is used to define an angular-axis. A chart can have more than one angular-axis positioned at different positions.
    radial-axis The radial tag is used to define a radial-axis. A chart can have more than one radial-axis positioned at different angles.
    barset The barset tag is used to define a bar data set. A chart can have more than one bar data set.
    scatterset The scatterset tag is used to define a scatter chart data set. A chart can have more than one scatter data set.
    scatter3vset The scatter3vset tag is used to define a scatter3v chart data set. A chart can have more than one scatter3v data set.
    legend The legend tag is used to define a legend.
    header The header of the chart.
    footer The footer of the chart.
