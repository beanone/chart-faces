Tag scatterset

  • Description:
    The scatterset tag is used to define a scatter chart data set. A chart can have more than one scatter data set.
  • Component Information 
    Info Value
    Component Type org.javaq.chartfaces.chart.part.scatterset
    Display Name None
    Handler Class None
    Renderer Type scatterset
    Description None
  • Attributes 
    Name Required Type Description
    axesIds false java.lang.Object ID values of the axes the scatterset is associated with. ID values are separated by comma.
    legendLabel false java.lang.Object Label of the legend for this scatterset.
    showTooltips false java.lang.Object A flag indicates whether tooltip text should be used for the data points.
    fill false java.lang.Object The filling property for the dot symbols of the scatter chart.
    lineStyle false java.lang.Object Style of the lines connecting the dot symbols if "showLine" is true.
    opacity false java.lang.Object The opacity of the fill of the symbols.
    showLine false java.lang.Object A flag indicates whether show lines in between the symbols.
    strokeColor false java.lang.Object The color of the strokes if "showLine" is true.
    strokeWidth false java.lang.Object The width of the strokes if "showLine" is true.
    symbol false java.lang.Object The name of the symbol. Valid values are "triangle", "circle", and "square".
    symbolSize false java.lang.Object The size of the symbol in the chart viewBox coordinates.
    value false java.lang.Object The data values. May be an org.javaq.chartfaces.api.IChartingData, an array, an java.util.Iterable, or a java.util.Map object.
